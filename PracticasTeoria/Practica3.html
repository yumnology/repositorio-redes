Práctica: Elementos de una Red Local


Resumen


En esta práctica, investigamos y configuramos diferentes redes locales (LAN) utilizando el software Packet Tracer de Cisco. Simulamos redes asignando direcciones IP a varias computadoras y utilizamos la función ping para verificar la comunicación entre ellas. Además, elaboramos tablas comparativas detallando la descripción y funcionamiento de los principales componentes de la red.

Introducción


Nos propusimos conocer y comprender los elementos fundamentales necesarios para configurar una red local y gestionar el acceso a los recursos de un equipo en red. Empleando el programa Packet Tracer de Cisco, simulamos diversas configuraciones de redes, asignando direcciones IP específicas y verificando la conectividad entre dispositivos. También, desarrollamos tablas comparativas para cada componente de red, proporcionando una visión detallada de su funcionamiento y aplicación en contextos reales.

Conclusiones


Al finalizar la práctica, logramos una comprensión sólida de los elementos clave de una red local y su configuración. Simulamos con éxito diversas redes, asignamos direcciones IP y confirmamos la comunicación efectiva entre computadoras mediante la función ping. Las tablas comparativas desarrolladas nos brindaron una clara comprensión de las funciones y aplicaciones de cada componente, fortaleciendo así nuestros conocimientos en la administración y diseño de redes locales.
