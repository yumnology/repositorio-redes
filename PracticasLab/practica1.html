Práctica: Wifi ESP32 Entradas y Salidas Digitales


Resumen

En esta primera abordamos la configuración y programación del módulo Wifi ESP32, enfocándonos en la comprensión y uso de sus entradas y salidas digitales. A través de una serie de pasos detallados, aprenderemos a configurar el entorno de desarrollo Arduino IDE, revisar el pinout y GPIO disponibles del ESP32, y desarrollar diversos programas para el control de un LED y el uso de botones.

Introducción


La práctica con el módulo Wifi ESP32 nos ofrece la oportunidad de familiarizarnos con las entradas y salidas digitales de este dispositivo. Esta guía nos proporciona las instrucciones necesarias para configurar el IDE de Arduino para el NodeMCU y ESP32, y desarrollar programas que nos permitan encender y apagar un LED mediante programación, retardos y el uso de botones. Además, se presenta una lista de materiales esenciales para llevar a cabo esta práctica de manera efectiva.

Materiales

  • ESP32 WROOM32
  • LED, resistencia de 220 Ohms
  • 4 Botones y 4 resistencias de 1K Ohms
  • Protoboard y cables

Desarrollo


Objetivo general


Conocer, identificar y comprobar el funcionamiento de las entradas y salidas digitales en el módulo Wifi ESP32.

Objeetivos específicos

  • Utilizando la información e instrucciones de la presentación, configurar el IDE de Arduino para poder programar el NodeMCU directamente desde el entorno de Arduino, instalando las herramientas necesarias:
    • http://arduino.esp8266.com/stable/package_esp8266com_index.json
    • https://randomnerdtutorials.com/installing-the-esp32-board-in-arduino-ide-windows-instructions/
  • Revisar el pinout y los GPIO disponibles para el uso del NodeMCU:
    • https://randomnerdtutorials.com/esp8266-pinout-reference-gpios/
    • https://randomnerdtutorials.com/esp32-pinout-reference-gpios/
  • Desarrollar un programa de encendido de un LED mediante programación.
  • Desarrollar un programa de apagado de un LED mediante programación.
  • Desarrollar un programa de encendido y apagado de un LED mediante retardos.
  • Desarrollar un sistema de encendido y apagado de un LED mediante el uso de botones.
  • Desarrollar un sistema de encendido y apagado de un LED mediante el uso de dos botones y una condicional OR.
  • Desarrollar un sistema de encendido y apagado de un LED mediante el uso de dos botones y una condicional AND.

Para esta práctica usamos la plataforma Wowki para desarrollar los ejercicios; posteriormente los hicimos en físico usando una ESP32 Wroom32.

Conclusiones


A través de esta práctica, adquirimos una comprensión sólida de cómo funcionan las entradas y salidas digitales en el ESP32. La configuración del entorno de desarrollo y la implementación de diversos programas nos permitieron no solo encender y apagar un LED, sino también interactuar con el módulo utilizando botones y condicionales. Esto nos proporcionó una base práctica que podemos aplicar a proyectos más complejos en el futuro, fortaleciendo nuestras habilidades en programación y manejo de hardware digital.
